# **FlappyBit**


FlappyBit turns Bitcoin chaos into a browser game where players dodge red candles, collect Sats, unlock eras, earn badges, and fly through crypto history.

__________________________________________________________________


## 🕹️ Play

https://flappybit.fun

(DM on X for beta access)

__________________________________________________________________


## 🚀 What it is

Crypto onboarding is still too financial, too technical, and too boring.

FlappyBit makes it playable.

Players fly through Bitcoin market eras, dodge red candles, stack soft Sats, unlock new eras, collect skins and avatars, and compete on weekly leaderboards.

## ✅ Current MVP

- Browser-based arcade gameplay.
- Bitcoin-history levels.
- Auth with email plus EVM, Solana, and Bitcoin wallets.
- Profiles, runs, invite codes, and leaderboards.
- Shop with eras, skins, avatars, and power-ups.
- Daily rewards, badges, and Sats progression.


## User loop:

Connect/login → claim starter Sats → play daily 
→ earn rewards → unlock eras → return tomorrow
